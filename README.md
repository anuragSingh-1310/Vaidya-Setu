# Vaidya Setu - AI-Powered Healthcare Platform (Frontend)

## About

Vaidya Setu is an AI-powered healthcare platform developed to simplify access to healthcare services through intelligent digital solutions. The platform combines AI, healthcare utilities, and communication features to provide users with a seamless healthcare experience.

This repository contains **only the frontend implementation** of the project. I worked as a **Frontend Developer** in a team where my responsibility was to design and develop the user interface using **HTML, CSS, and JavaScript**. The backend, AI models, and integrations were developed by other team members and are not included in this repository.

---

## My Contribution

As a Frontend Developer, I was responsible for:

- Designing and developing responsive web pages using HTML, CSS, and JavaScript.
- Building the landing page and multiple application screens.
- Developing Login and Signup interfaces.
- Creating About, Contact, and Services pages.
- Implementing interactive UI components and client-side functionality.
- Collaborating with backend and AI teams to integrate frontend interfaces with APIs.

---

## Features of the Complete Project

Although this repository only contains the frontend, the complete Vaidya Setu platform included the following features:

### 🧠 AI Cancer Detection
- Upload medical images for AI analysis.
- Predicts whether cancer is detected.
- Displays the predicted cancer category.

### 🤖 AI Health Chatbot
- Answers common healthcare-related questions.
- Provides basic medical guidance and health information.

### 📹 Video Consultation
- Secure video calling interface for online doctor consultations.

### 📄 Medical Report Summarizer
- Generates simplified summaries of medical reports using AI.

### 📅 Google Calendar Integration
- Add appointments, medication reminders, and medical events directly to Google Calendar.

## Getting Started

1. Clone the repository

```bash

```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional installation or dependencies are required.
